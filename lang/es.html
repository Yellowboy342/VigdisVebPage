SIMPLIFICA LA VIDA DIARIA

Estructura visual para rutinas diarias

Una aplicación de horarios visuales intuitiva y local, diseñada por profesionales y padres para apoyar a individuos y familias brindando una visión clara de las actividades diarias.

Vigdís es una herramienta para cualquiera que desee mayor previsibilidad. Vigdís está diseñada para satisfacer las diversas necesidades de las personas y es útil tanto para quienes utilizan una amplia gama de servicios como para todos los demás.
